## Policy

- [ ] Verification date updated.
- [ ] Pricing policy set to source-link behavior (no guaranteed quotes).

## Prompt

- [ ] Quick Pick + Guided + Allergen-first flows confirmed.

## QA
- [ ] Transcript suite cases updated for restaurant specifics.
- [ ] All safety and citation cases pass.
- [ ] Final report generated and saved.

## Publication

- [ ] README includes reproducible setup steps.
